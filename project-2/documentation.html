Noteworthy

  • Comets: Added comets that fall down the screen using JavaScript classes.
  • Fetch: Asynchronously fetches JSON from a file for presets.
  • Web Components: Used web components for header, footer, and navbar.
  • Transformations: Used transformations to make the bars go in a circle.

Grading

  • Navigation: Tells the user which page they are currently on.
  • Bulma: Uses the Bulma CSS framework.
  • Labels: Easy to use with instructions, buttons and inputs are all labelled.
  • Error Handling: Handles errors gracefully, though no errors appear in testing.
  • App States: App tells you when song is playing.
