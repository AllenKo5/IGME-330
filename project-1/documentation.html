Functionality

Users will be able to search for specific cards using a series of parameters, including card name, card supertype/subtype, Type, Attribute, Level/Rank, ATK/DEF, etc. These cards can then be selected to display information about them, as well as being able to favorite them and save them to a community page displaying the most favorited cards.
